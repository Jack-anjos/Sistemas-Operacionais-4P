
Instruções para Compilar e Executar os Programas:

1. Como compilar os programas:
   Para compilar qualquer um dos programas, use o comando gcc. Por exemplo, para compilar o programa exercicio01.c, use:
   gcc -o exercicio01 exercicio01.c

   Repita o mesmo processo para exercicio02.c e exercicio03.c:
   gcc -o exercicio02 exercicio02.c
   gcc -o exercicio03 exercicio03.c

2. Como executar os programas:
   Após a compilação, execute os programas usando o seguinte comando:
   ./exercicio01
   ./exercicio02
   ./exercicio03

3. Como comprovar que os resultados propostos foram alcançados:
   - exercicio01.c: Este programa lê e exibe o conteúdo do arquivo 'lab2_ex1.txt'. Confira a saída no terminal para verificar o conteúdo lido.
   - exercicio02.c: Este programa conta e exibe o número de linhas no arquivo 'lab2_ex2.txt'. Confira a saída no terminal para verificar o número de linhas.
   - exercicio03.c: Este programa copia o conteúdo do arquivo 'lab2_ex3_origem.txt' para 'lab2_ex3_destino.txt'. Confira os conteúdos do arquivo destino para comprovar que a cópia foi realizada corretamente.

Nota: Certifique-se de que os arquivos 'lab2_ex1.txt', 'lab2_ex2.txt', 'lab2_ex3_origem.txt', e 'lab2_ex3_destino.txt' estão presentes no diretório de trabalho antes de executar os programas.
